Fuji

Celeste 64 Mod Loader

Fuji is a free and open souce mod loader for Celeste 64: Fragments of the Mountain.

Features

  • Create and download mods which alter the game or expand it with custom entities, features, and anything else!
  • Install and make custom maps
  • Support for mods to replace the game's default assets

Instructions

Click the "Install latest release" button at the top of the page to be taken to the latest release of Fuji.

Download the zip file and unzip it, then open the executable to start playing!
