Fuji

Celeste 64 Mod Loader

Fuji is a free and open source mod loader for Celeste 64: Fragments of the Mountain.

Get the latest release Find mods on GameBanana

Features

  • Create and download mods which alter the game or expand it with custom entities, features, and anything else!
  • Install and make custom maps
  • Support for mods to replace the game's default assets

Instructions

Click the "Get the latest release" button at the top of the page to be taken to the latest release of Fuji.

Download the zip file and unzip it, then open the executable to start playing!


To install mods, download the zip file you got from the mod author, and place it in the game's Mods folder. Unzipping it is optional.
